# Tgreys_Old_Cabbages
Legacy collection of my standalone Cabbage effects and instruments

These *should* work across all platforms, but have only been tested in OSX
with Csound version 6.14 (4.21.2020) and Cabbage beta 2.3.39

Included Effects
	Delayed2.csd
	DoubleDown.csd
	Flanged.csd
	ShapedUp.csd
	SolinaChorus.csd
	Tubewarmth.csd

Included Synths
	TheKicker.csd
